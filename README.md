## 📐 Design System

### Color Palette

| Color | HSL Value | Usage |
|:-----:|-----------|-------|
| 🟣 Primary | `hsl(250, 90%, 62%)` | Buttons, links, focus states |
| 🔴 Accent | `hsl(330, 85%, 60%)` | Gradient endpoints, highlights |
| ⚫ Background | `hsl(230, 25%, 7%)` | Page background |
| ⬛ Elevated | `hsl(230, 18%, 13%)` | Cards, modals |
| 🟢 Success | `hsl(150, 75%, 50%)` | Validation, success states |
| 🔴 Error | `hsl(0, 80%, 60%)` | Error states, warnings |

### Typography

| Font | Weight | Usage |
|------|--------|-------|
| **Space Grotesk** | 700–800 | Headlines, brand text |
| **Inter** | 300–600 | Body text, labels, inputs |

## 📝 Form Fields Reference

### Sign-Up Form (3 Steps)

<details>
<summary><strong>Step 1 — Personal Information</strong></summary>

| Field | Type | Validation |
|-------|------|------------|
| First Name | `text` | Required, min 2 chars |
| Last Name | `text` | Required, min 2 chars |
| Username | `text` | Required, 3-20 chars, alphanumeric + underscore |
| Date of Birth | `date` | Required, must be 13+ years old |
| Gender | `radio` | Required, Male / Female / Other |

</details>

<details>
<summary><strong>Step 2 — Contact Details</strong></summary>

| Field | Type | Validation |
|-------|------|------------|
| Email | `email` | Required, valid email format |
| Phone | `tel` | Optional |
| Country | `select` | Required, 12+ country options |
| Address | `textarea` | Optional |

</details>

<details>
<summary><strong>Step 3 — Security</strong></summary>

| Field | Type | Validation |
|-------|------|------------|
| Password | `password` | Required, 8+ chars, uppercase, lowercase, number, special char |
| Confirm Password | `password` | Required, must match password |
| Terms & Conditions | `checkbox` | Required |
| Newsletter | `checkbox` | Optional |

</details>

### Login Form

| Field | Type | Validation |
|-------|------|------------|
| Email / Username | `text` | Required, min 3 chars |
| Password | `password` | Required |
| Remember Me | `checkbox` | Optional |
